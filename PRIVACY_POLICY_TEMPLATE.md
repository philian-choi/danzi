# 개인정보 처리방침 (Privacy Policy)

**단지노트 앱 개인정보 처리방침**

## 1. 수집하는 정보

단지노트 앱은 다음 정보를 수집합니다:

### 1.1 사용자 생성 콘텐츠
- **임장 기록 데이터**: 단지명, 소재지, 연식, 구조, 시세 정보, 메모 등 사용자가 직접 입력하는 모든 정보
- **사진**: 사용자가 촬영하거나 선택한 임장 관련 사진
- **날짜 및 시간 정보**: 임장 기록 작성 날짜 및 시간

### 1.2 수집하지 않는 정보
- 개인 신원을 식별할 수 있는 정보
- 위치 정보 (GPS) - 사용자가 직접 텍스트로 입력하는 주소 정보만 수집
- 연락처 정보
- 결제 정보
- 기기 고유 식별자

## 2. 정보 저장 위치 및 방법

- **저장 위치**: 모든 데이터는 사용자의 기기 내부 저장소에만 저장됩니다
- **저장 방식**: 
  - 텍스트 데이터: iOS 앱의 UserDefaults(SharedPreferences)에 암호화되지 않은 JSON 형식으로 저장
  - 사진 파일: 앱의 문서 디렉터리에 저장
- **클라우드 저장**: 사용하지 않습니다. 모든 데이터는 로컬 기기에만 저장됩니다.

## 3. 정보 사용 목적

수집한 정보는 다음 목적으로만 사용됩니다:
- 임장 기록의 작성, 저장, 수정, 삭제
- 기록 검색 및 정렬 기능 제공
- 데이터 백업 및 복원 기능 제공

## 4. 정보 공유 및 제공

- **제3자 공유**: 어떠한 제3자와도 정보를 공유하지 않습니다
- **광고 및 마케팅**: 사용하지 않습니다
- **트래킹**: 사용하지 않습니다
- **분석 서비스**: 사용하지 않습니다

## 5. 데이터 백업 및 복원

- 앱 내에서 JSON 형식으로 데이터를 내보낼 수 있습니다
- 내보낸 데이터는 사용자가 선택한 방식(이메일, 메시지, 클라우드 저장소 등)으로 저장됩니다
- 데이터 복원 시 사용자가 선택한 JSON 파일을 불러와 앱 내부 저장소에 저장합니다
- **참고**: 백업 파일에는 텍스트 데이터만 포함되며, 사진 파일은 별도로 관리되어야 합니다

## 6. 데이터 삭제

사용자는 다음 방법으로 데이터를 삭제할 수 있습니다:
- 앱 내에서 개별 기록 삭제
- 앱 삭제 시 모든 데이터가 기기에서 완전히 삭제됩니다

## 7. 앱 권한

앱이 요청하는 권한과 그 목적:

- **카메라 권한**: 임장 사진을 촬영하기 위해 필요합니다
- **사진 라이브러리 접근 권한**: 갤러리에서 사진을 선택하기 위해 필요합니다

## 8. 보안

- 모든 데이터는 사용자 기기에만 저장되며 네트워크를 통해 전송되지 않습니다
- 데이터는 앱 내부 저장소에 평문으로 저장됩니다
- 기기 잠금 및 백업 보안은 사용자 기기의 기본 보안 설정에 따라 달라집니다

## 9. 아동의 개인정보 보호

이 앱은 13세 미만의 아동을 대상으로 하지 않으며, 아동의 개인정보를 의도적으로 수집하지 않습니다.

## 10. 개인정보 처리방침 변경

개인정보 처리방침은 법령, 정책 또는 보안기술의 변경에 따라 내용의 추가, 삭제 및 수정이 있을 수 있습니다. 변경사항은 앱 내 공지 또는 웹사이트를 통해 안내드립니다.

## 호스팅 가이드

이 문서를 웹에 호스팅하려면:

1. **GitHub Pages** (무료, 추천):
   - GitHub 저장소 생성
   - `docs/privacy-policy.md` 또는 `privacy-policy.html` 파일 생성
   - Settings > Pages에서 GitHub Pages 활성화
   - URL 예: `https://[username].github.io/[repository]/privacy-policy.html`

2. **웹사이트 호스팅**:
   - 기존 웹사이트가 있다면 `/privacy-policy` 경로에 업로드
   - URL 예: `https://yourdomain.com/privacy-policy`

3. **App Store Connect에 입력**:
   - App Store Connect > 앱 > 앱 정보 > 프라이버시 정책 URL에 위 URL 입력
